📋 Embed Code: {{ client_id }}

Copy and paste this code into your client's website to add the chat widget.

Step 1: Basic Embed (paste before </body>)

This adds a floating chat bubble to the bottom-right of any page.

<script
  src="{{ request.host_url }}static/widget.js"
  data-client="{{ client_id }}">
</script>

Step 2: Customized (optional)

Change the color, title, or position of the widget.

<script
  src="{{ request.host_url }}static/widget.js"
  data-client="{{ client_id }}"
  data-color="#4f46e5"
  data-title="Ask Our AI"
  data-position="right">
</script>

WordPress Instructions

For WordPress sites:

  1. Go to Appearance → Theme File Editor
  2. Open footer.php
  3. Paste the code just before </body>
  4. Click Update File

Or use a plugin like "Insert Headers and Footers" to add it without editing theme files.

Note: The widget connects to your API at {{ request.host_url }}{{ client_id }}/ask — make sure the Render deployment is running.
